INSTRUCTIONS - MASS PAYMENTS WITH .TXT - .CSV OR .SDV FILE

The file format should be .TXT - .CSV or .SDV (MS-DOS). This file(s) can be easily generated from most spreadsheet applications such as Excel.

There is a limit of 1,000 payments per file. If you wish to send more than 1,000 payments please split your main file in smaller ones as many as necessary so each has a maximum of 1,000 payments. Total amount per file limit is $200,000.00.

The file should contain dollar amount in the first column, username or email in the second and information to the recipient's in the third column. All amounts should be greater than $1.00.
